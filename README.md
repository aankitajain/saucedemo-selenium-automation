# Selenium SauceDemo Automation

This project automates a simple user journey on SauceDemo.

User Journey:
Login → Open product → Add to cart → Back to products → Open another product → Add to cart → View cart

Validations on Cart Page:
- Remove button exists for products
- Continue Shopping button exists
- Checkout button exists
